# 🌸 Iris Species Classification

Interactive dashboard for classifying Iris flower species using machine learning.

**Course:** Data Mining — Universidad de la Costa (CUC) 2026-1  
**Professor:** José Escorcia Gutierrez, Ph.D.

## Authors

- Rafael Ricardo Romo Restrepo

## Description

This project implements an end-to-end data mining pipeline for the Iris species classification problem. The dashboard allows users to explore the dataset, compare multiple classification models, and predict flower species in real time with 3D visualization.

## Features

- **Workflow & Methodology:** Step-by-step explanation of the data mining pipeline
- **Model Metrics:** Accuracy, Precision, Recall, F1-Score, Confusion Matrix, Cross-Validation
- **Species Prediction:** Interactive input panel with 3D PCA scatter plot
- **Data Exploration:** Histograms, Box Plots, Scatter Matrix, Correlation Heatmap, Violin Plots
- **Model Comparison:** Random Forest, SVM, KNN, Gradient Boosting
